# Lighthouse

> Every grad programme, one place, never stale.

A grad job tracker for students in Ireland and the UK. Curated, freshness-verified directory of grad programmes plus a personal application pipeline.

Building in public — week 0.

## Stack

- Go (backend API + worker)
- Next.js + TypeScript (frontend)
- PostgreSQL

## Status

Pre-alpha. Setting up the foundations.
